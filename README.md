### Spotlight 是什么？

我相信所有模拟城市玩家都会遇到在 Simtropolis 、 SC4 Devotion 和 模拟城市中文网 搜索插件的经历，我也是如此。所以 Spotlight 就诞生了，一个可以同时搜索以上网站插件内容的搜索引擎，你可以输入中文描述或是插件名字进行搜索（感谢汽水提供语义描述支持），搜索结果会自动弹出，单击即可进入插件下载页面。你可能需要登录才能下载，这不在 Spotlight 的能力范畴 :)
